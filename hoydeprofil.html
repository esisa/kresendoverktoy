Kresendo høydeprofil

Når du trenger en høydeprofil nå!

Ved å bruke Kresendos høydeprofil kan du legge til høydeprofiler på ditt eget nettsted. Prøv høydeprofilen her eller se hvordan det ser ut på Turkompisen.no

Høydeprofilen gir deg svar på GeoJSON-format og det er veldig enkelt å bruke tjenesten. Send inn data med HTTP POST i GeoJSON eller WKT format. Se eksempelet for hvordan du kan gjøre det med jQuery.

For å teste kan du bruke curl. Eksempel på kall er:

curl -H "Content-type: application/json" -X POST http://verktoy.kresendo.no/elevProfile/elevationprofile.json -d @example.geojson

eller

curl -H "Content-type: text/plain" -X POST http://verktoy.kresendo.no/elevProfile/elevationprofile.wkt -d @example.wkt

Eksempler på GeoJSON eller WKT du kan bruke finner du her

Bruker du Kresendo høydeprofil må du tydlig vise at det er Kresendo som tilbyr tjenesten.

Kresendo høydeprofil er også tilgjengelig som fri programvare. Se på koden og prøv den ut selv på Github. Ønsker du tilpasninger eller bruke høydeprofilen i større skala så ta kontakt.

Deler av høydeprofil-koden ble først utviklet i samarbeid med Vestlandsforsking til deres Interreg-prosjekt TG4NP.

Merk at Kresendo høydeprofil bruker Aster data som for Norge kan mangle data i enkelte mindre områder. Tjenesten leverer negative høydeverdier for disse områdene. Du må sjøl håndtere dette. Se koden til eksempelet for å se hvordan dette kan løses. Se Turkompisen.no for hvilke områder som mangler data.
